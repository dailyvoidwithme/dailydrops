# DailyDrops - Premium Content Platform

A beautiful, eye-catching website for premium content subscriptions with an integrated admin panel.

## Features

- 🎨 Eye-catching animated gradient background
- 📱 Fully responsive design
- 💳 Subscription options (₹99 Normal | ₹189 Ultra Premium)
- 📝 User details collection (Telegram, WhatsApp)
- 📊 Admin dashboard with statistics
- 🖼️ Thumbnail management (Add/Edit/Delete)
- 💰 Transaction tracking
- 🔒 Secure admin login
